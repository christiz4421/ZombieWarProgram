Description
This program simulates a zombie virus outbreak where a group of survivors must make it to a safe place without getting killed by the zombies. 
Each survivor or zombie character has unique health and attack values that determine how they perform in combat.

At this point, the Character class has been completed and provides the core attributes and behaviors that all survivors and zombies will inherit.

Features
Character Class
• 	Defines shared attributes for all characters
• 	Stores health and attack values
• 	Prevents dead characters from attacking
• 	Ensures health never drops below zero
• 	Includes full Javadoc and inline comments

Files Included
• 	Character.java, Which is the Abstract base class for all survivors and zombies

How to Run

2. 	Open the project in IntelliJ IDEA or any Java IDE
